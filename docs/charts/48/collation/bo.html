The following illustrates the ordering for the Tibetan collation tailorings. It does not show the strength differences, such as where case is ignored where there are letter differences. The search order is special: it only used for comparing characters for similarity, so the order among the characters does not matter. Where a type is not present, such as emoji or search, it defaults to the Root type.

  • The characters used in the illustration are:
    1. those not tailored (added from standard exemplars for context)
    2. those tailored, but not in any exemplars (standard, aux, punctuation)
    3. those both tailored and in exemplars
  • The tailored characters may include:
    1. some longer strings (contractions) from the rules
    2. generated Unicode characters (for canonical closure)
